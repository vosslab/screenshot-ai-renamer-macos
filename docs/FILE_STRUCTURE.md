# File structure

## Root
- `AGENTS.md`: agent pipeline overview and coordination notes.
- `Brewfile`: Homebrew dependencies.
- `LICENSE`: project license.
- `README.md`: project overview and doc links.
- `pip_requirements.txt`: Python dependencies.
- `screenshot-renamer.py`: main CLI entrypoint.
- `docs/`: documentation.
- `tools/`: OCR, captioning, metadata, and model helpers.

## Docs
- `docs/CHANGELOG.md`: user-facing change history.
- `docs/INSTALL.md`: setup steps and environment requirements.
- `docs/USAGE.md`: CLI usage and examples.
- `docs/TROUBLESHOOTING.md`: known issues and fixes.
- `docs/PYTHON_STYLE.md`: Python coding conventions.
- `docs/MARKDOWN_STYLE.md`: documentation writing conventions.
- `docs/REPO_STYLE.md`: repository conventions.
